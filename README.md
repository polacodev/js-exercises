### JavaScript Exercises

#### Clone the Repository
```sh
git clone https://github.com/polacodev/js-exercises.git
```

#### Install the dependencies
```sh
npm install
```

#### Run the Exercises
```sh
npm start
```

#### Run Test
```sh
npm jest
```
